欢迎使用 CSS Inspector

这是一个专门设计的测试页面，包含各种不同的 CSS 样式和布局。

尝试以下操作：

  • 点击浏览器扩展图标打开 CSS Inspector
  • 点击 "Start Picking" 开始选择元素
  • 鼠标悬停并点击任何元素
  • 查看和修改 CSS 属性
  • 导出你的修改
